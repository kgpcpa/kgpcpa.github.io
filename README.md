# KGPCPA.COM

This README describes how to build the kgpcpa.com website.

Detailed instructions at: https://help.github.com/articles/using-jekyll-with-pages/


System requirements.

  ruby >= 1.9.2
  jekyll >= 2.0 (http://jekyllrb.com/) 


Setup instructions

  run the command `jekyll serve --watch`
